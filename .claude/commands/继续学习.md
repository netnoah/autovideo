# 继续学习 - 广东干部培训网络学院

自动完成以下流程：
1. 关闭已有的 Edge 浏览器进程
2. 用 Playwright 启动 Edge（复用登录态）
3. 打开学习中心，点击"继续学习"
4. 等待视频页面加载，自动点击播放

## 执行步骤

### Step 1: 关闭 Edge

```bash
taskkill //F //IM msedge.exe 2>&1 || echo "No Edge process"
```

### Step 2: 运行自动学习脚本

在项目根目录运行 `./auto-study.js`：

```bash
node auto-study.js 2>&1
```

脚本会自动完成：
- 启动 Edge 并加载用户 Cookie（复用已登录的 Microsoft Edge 用户数据目录）
- 导航到 `https://gbpx.gd.gov.cn/gdceportal/study/StudyCenter.aspx`
- 在课程列表 iframe 中找到"继续学习"按钮并点击
- 等待新标签页打开视频播放页
- 在 iframe 内找到 `.vjs-big-play-button`（Video.js 播放按钮）并点击
- 截图保存到 `./video-playing.png`

### Step 3: 报告结果

脚本输出完成后，读取截图确认视频是否在播放：

- 如果脚本输出 `Video playback started!`，告诉用户视频已开始播放
- 如果失败，查看错误信息并告知用户

## 注意事项

- Edge 必须已登录该网站（脚本复用 Edge 的 User Data 目录中的 Cookie）
- 如果提示登录过期，需要用户手动在 Edge 中重新登录一次
- 脚本运行期间不要手动操作 Edge
- 需要已安装 Node.js 和 Playwright（`npm install playwright` + `npx playwright install chromium`）
